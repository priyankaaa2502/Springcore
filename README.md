# Spring Core Projects

This repository contains multiple hands-on projects demonstrating core concepts of the Spring Framework. These projects are created to build a strong foundation in Spring and understand how different features work internally.

Projects included

Dependency Injection
Demonstrates how Spring manages object creation and dependency resolution. Covers constructor and setter injection.

Value Annotation
Shows how to inject values from properties files. Useful for external configuration handling.

Autowired
Demonstrates automatic dependency injection. Covers field, constructor, and setter injection.

Bean Lifecycle
Explains initialization and destruction of Spring beans. Covers lifecycle annotations and interfaces.

Spring Common Annotations
Covers commonly used annotations such as Component, Service, Repository, and Controller.

Technologies used
